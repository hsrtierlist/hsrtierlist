# HSRTierList

> Honkai: Star Rail Tier List & Strategy Guide

A comprehensive resource for HSR players featuring character rankings, team recommendations, and banner pull advice.

**Live at**: https://hsrtierlist.net

---

## Features

- **Character Tier List** - Ranked by role and element with detailed evaluations
- **Team Rankings** - Meta team compositions for various content
- **Banner Advice** - Pull recommendations for current and upcoming banners
- **Endgame Guides** - Strategy references for challenging content
